dev.plone.org has moved!

Plone's version control, issue tracking, and documentation have been retooled.

To file or view an issue please visit the Plone Issue Tracker on GitHub.

Plone documentation — how to install, manage, use, and develop for Plone — is available at docs.plone.org.


However, if you want to contribute to Plone Core you'll need to read our Process for Plone Core Development first.

A read-only version of the old Subversion repository is available at svn.plone.org. To download it (it's ~16 GB) visit archive.plone.org/svn.

The old static version of dev.plone.org, including old PLIPS and open issues, can be downloaded from archive.plone.org.
